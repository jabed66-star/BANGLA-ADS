Publisher Endpoints

GET https://api.adsbangla.com/v1/publisher/websites

অনুমোদিত ওয়েবসাইটগুলোর তালিকা পেতে এই এন্ডপয়েন্ট ব্যবহার করুন।

{ "status": "success", "data": [ { "id": 1, "domain": "example.com", "status": "approved" } ] }

Advertiser Endpoints

POST https://api.adsbangla.com/v1/advertiser/campaigns/create

নতুন ক্যাম্পেইন তৈরি করার এন্ডপয়েন্ট।

{ "campaign_name": "Summer Special", "budget": 100.00, "ad_type": "banner" }

Statistics Endpoints

GET https://api.adsbangla.com/v1/stats/daily

দৈনিক ইম্প্রেশন, ক্লিক এবং আর্নিং রিপোর্ট ডাটা।

{ "date": "2026-07-21", "impressions": 15000, "clicks": 450, "earnings": 12.50 }

Payment Endpoints

POST https://api.adsbangla.com/v1/payment/withdraw/request

উইথড্র রিকুয়েস্ট সাবমিট করার API এন্ডপয়েন্ট।

{ "amount": 50.00, "method": "bkash", "account_no": "01700000000" }
