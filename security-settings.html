Security & Defense Controls

Application Shields & Protection Protocols

Force HTTPS for all web traffic and secure cookies.
Active (TLS 1.3)
Validate CSRF tokens for all form requests & actions.
Sanitize user inputs & enable Content Security Policy (CSP).
Prepared statements & ORM query escaping filter enabled.

Google reCAPTCHA Protection

Prevents automated bot login & sign-up attempts.

Email & Phone Authentication

Require new users to verify email before account activation.
Allow users to log in using One-Time Password via Mobile SMS.

Two-Factor Authentication (2FA) Policies

Require Authenticator App (Google Auth / Authy) for Admin accounts.
Allow users to enable 2FA on their profiles for security.
